Bind user to taxonomy and display only posts on selected taxonomy.

== Description ==
The admin can restrict posts by selected users to a selected taxonomy. Restricted users won't view the taxonomy selection panel in edit screens and in taxonomy list. 

== Installation ==
Use standard word press installation
